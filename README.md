# AGV + MELFA Robot Arm Integration

Integrating Keyence PLC AGV (ROS2) with Mitsubishi MELFA RV4-FL robot arm (ROS1).

## 📦 Dependencies

This project uses [melfa_robot](https://github.com/tork-a/melfa_robot) by tork-a (Apache 2.0).

## 🚀 How to Run

Install ROS2 system for AGV, and install ROS1 in your docker or another PC.
When you run the software, you must run the bridge program first.
